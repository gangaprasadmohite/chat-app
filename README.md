# ChatApp

End-to-end chat application using MERN Stack

# Installation

1. download the repo
2. open terminal with in the repo directory
3. install and run the server-side:

```
cd server
npm i
npm start
```

4. install and run the client-side:

```
cd client
npm i
npm start
```

5. open the (http://localhost:3000/) on your browser
